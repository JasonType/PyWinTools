A Tool that makes geting Info Esay for Windows 10 / 11

## Features

- IP Lockup
- Proxy Scrape
- Show IP
- WiFiPass
- IP Pinger
- DDoS Tool

### Batch Version
- Windows 10 / 11

### Python Version
- Python 3.10 or higher
- Windows 10 / 11
