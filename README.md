# WinDevTools
A Tool that makes geting Info Esay for Windows 10 / 11

## Features

- IP Lockup
- Proxy Scrape
- Show IP
- WiFiPass
- IP Pinger
- DDoS Tool

## Versions
- [Batch](https://github.com/JasonType/WinDevTools/tree/Batch)
- [Python](https://github.com/JasonType/WinDevTools/tree/Python)

## Requirements

### Batch Version
- Windows 10 / 11

### Python Version
- Python 3.10 or higher
- Windows 10 / 11
